# Sistema de Registro de Notas

# Descripcion
Programa en C++ que permite registrar estudiantes, guardar sus notas, 
calcular promedios y mostrar un resumen general.

## Funcionalidades
- Registrar estudiantes
- Guardar 3 notas por estudiante
- Calcular promedios
- Mostrar listado de estudiantes
- Buscar estudiante por nombre
- Mostrar resumen general
- Clasificar el rendimiento del estudiante
